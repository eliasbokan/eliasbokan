## 🧠 Unsupervised Learning Project

This project explores **unsupervised machine learning** techniques, focusing on **K-Means clustering** and **DBSCAN** implemented in both **Scikit-Learn** and **TensorFlow**.  
The goal was to understand how different clustering algorithms perform across synthetic and real-world datasets, using evaluation methods like the **Elbow method**, **Silhouette score**, and **Adjusted Rand Index (ARI)**.

Key outcomes include:
- High-quality cluster formation using K-Means (Silhouette score: 0.76)
- Implementation of K-Means from scratch in TensorFlow for deeper algorithmic understanding
- Application of DBSCAN on the Iris dataset, achieving moderate alignment with true labels (ARI = 0.54)
- Visual insights into clustering boundaries, centroids, and data separability

This project highlights both the **theoretical understanding** and **practical application** of unsupervised learning methods in Python.
